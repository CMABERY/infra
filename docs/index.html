Codex-Like MVP Project

This project contains a minimal transaction-first coding agent kernel.

Features

  • JSON-RPC-ish transport over stdio (NDJSON)
  • Typed turn/item stream
  • Approval gating for commands + file changes
  • Patch proposal + apply
  • Append-only ledger
  • Multi-file patchsets
  • Tool-call provenance artifacts
  • Invariants validator

Quick Start

python -m codex_like_mvp.client.mvp_cli demo
